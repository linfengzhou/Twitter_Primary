Election 2016 Sentiment Analysis


ABOUT

This dashboard is designed to visualize what twitter users are talking about for the 2016 presidential election and their preference towards different candidates based on real-time twitter data. All visualizations would be updated every 10 minutes. There are two sections of the dashboard: corpus exploration and sentiment analysis. In the corpus exploration tab, a tree map is used to visualize the composition and percentage of high frequently mentioned words on twitter. In addition, a network was built to display the connection between keywords and candidates. In the sentiment analysis tab, the results of sentiment analysis on tweets were visualized through a scatter plot. The two axes correspond to the two dimensions that we used for sentiment analysis: polarity and objectivity. Also, a bar chart was used to visualize the polarity of tweets particularly.

Each point in the scatter plot represents one tweet, with the sentiment analysis results as the coordinates: x-axis value for objectivity (from 0.0 to 1.0) and y-axis value for polarity (from -1.0 to 1.0). Similar as above, the color of each point stands for the candidate that this tweet corresponds to. When hovering on a point, the corresponding tweet would be showed in a tooltip.
